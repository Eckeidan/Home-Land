# Onboarding Visual Prototype

This dependency-free prototype validates the onboarding hierarchy, interaction
model, responsive layout, and visual direction before Next.js implementation.

Run locally from the repository root:

```bash
python3 -m http.server 4173 --directory prototypes/onboarding
```

Then open `http://127.0.0.1:4173`.

The prototype does not transmit or persist entered data. Example names and
addresses are demonstration content only.
